# Сканер фьючерсного рынка — простая версия на русском

Это приложение не показывает сухие индикаторы. Оно само просматривает **ликвидные USDT-фьючерсы Bybit**, ищет лучшие long-идеи и показывает их в простом виде:

- какую монету купить;
- покупать сейчас или ждать;
- где вход;
- где стоп;
- где продавать частично и полностью.

## Что внутри

- **backend** на Node.js + Express;
- **frontend** на React + Vite;
- фоновый сканер рынка;
- хранение сигналов в JSON;
- необязательный слой **OpenAI** для дополнительного русскоязычного разбора.

## Как работает логика

1. Backend получает список фьючерсных инструментов Bybit (`linear`) и отбирает только ликвидные монеты.
2. Для каждой монеты считает EMA20 / EMA50 / EMA200, RSI, ATR, ADX, моментум и объём.
3. Находит три состояния:
   - **Покупать сейчас**
   - **Ждать**
   - **Не покупать / выходить**
4. Для хороших идей строит торговый план:
   - зона входа
   - стоп
   - продажа 1
   - продажа 2

## Интерфейс

На главном экране есть:

- **Лучшая идея сейчас**
- **Готовые идеи**
- фильтр по типу сигнала
- поиск по монете
- история сигналов

## Переменные окружения

Используйте `.env.example`.

Главные параметры:

- `MARKET_TIMEFRAMES=15,60` — какие таймфреймы сканировать
- `MAX_SYMBOLS_TO_ANALYZE=40` — сколько ликвидных монет брать в один цикл
- `MIN_TURNOVER_24H_USD=2000000` — минимальный оборот 24ч для попадания в сканер
- `MAX_SPREAD_PCT=0.45` — максимальный спред
- `ACCOUNT_SIZE_USD=100` — размер депозита для расчёта риска
- `RISK_PER_TRADE_PCT=1` — риск на одну идею

Если хотите ИИ-разборы, добавьте:

- `OPENAI_API_KEY`
- `AI_ANALYSIS_ENABLED=true`

Без ключа приложение тоже работает: просто без блока анализа от ИИ.

## Локальный запуск

```bash

npm run build --workspace server
npm run build --workspace webapp
npm start
```

- `http://localhost:3001/health`
- `http://localhost:3001/api/overview`
- `http://localhost:3001/`

## Важная оговорка

Это не автоторговля. Приложение даёт **торговые идеи и план**, но сделки вы открываете сами.
